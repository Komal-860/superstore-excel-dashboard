🛒 Superstore Sales Analysis — Excel Dashboard Project

📊 Project Overview
An end-to-end sales analysis of a US-based retail superstore, built entirely in Microsoft Excel using Pivot Tables, Charts, Slicers, and KPI Cards across 4 interactive dashboards.

🗂️ Dashboard Pages
#DashboardKey Focus1Executive SummaryKPIs, Sales trend, Region & Year breakdown2Product & Category AnalysisSub-category profit, Discount vs Profit scatter3Regional & State PerformanceUS Map visual, Top cities, Region share4Customer & Segment AnalysisSegment profit, Ship mode, Monthly orders

📈 Key Insights

💰 Total Sales: $2,297,201 across 9,994 orders (2014–2017)
📦 Most Profitable Sub-category: Copiers ($55,618 profit)
⚠️ Biggest Loss Maker: Tables (26.13% avg discount → net losses)
🌍 Top Region: West (32% of sales), led by California
👥 Best Profit/Order Segment: Home Office
📅 Peak Period: Q4 (Nov–Dec) every year consistently
